# erratum

Personal portfolio site built with React, Three.js, and Framer Motion.

## Stack

- **React** — component-based UI
- **Three.js** — 3D animated hero background
- **Framer Motion** — scroll-triggered animations
- **Vite** — build tooling
- **CSS** — custom styling, no UI framework

## Sections

- **Hero** — animated line waves with a typewriter subtitle
- **About** — bio and skills
- **Projects** — Weather App, Job Tracker, and this portfolio
- **Contact** — email, GitHub, LinkedIn, and resume links

## Getting Started

```bash
cd portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```
